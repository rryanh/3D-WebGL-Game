## Getting Started

To get a local copy up and running follow these example steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

2. Clone the repo
   ```sh
   git clone https://github.com/rryanh/3D-WebGL-Game.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
